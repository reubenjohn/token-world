---

## Sampling Rate

- **After every task commit:** Run `uv run pytest tests/test_operator -x -q`
- **After every plan wave:** Run `uv run pytest -v` (skip `slow` marker by default)
- **Before `/gsd-verify-work`:** Full suite including `slow` marker must be green
- **Max feedback latency:** 30 seconds for quick tests; 3 minutes for slow integration tests

---

## Test-Infra Decisions (seeded from RESEARCH.md)

- **Real-LLM integration tests** are the default (D-13). Mark them `@pytest.mark.slow` so local dev can skip; CI runs them on nightly cadence (planner confirms).
- **No VCR in v1.** Revisit if slow-suite duration exceeds ~5 minutes total.
- **Engine stub** lives at `src/token_world/operator/testing.py` (research A3) — harness integration tests import it; production code does not.
- **Agent SDK cost cap:** document expected $/test in the first real integration test's docstring so cost drift is visible in PRs.
