   ```python
   def load_run_status(universe_dir: Path) -> dict[str, Any]:
       pid_path = universe_dir / ".run-pid"
       if not pid_path.exists():
           return {"state": "idle", "pid": None, "started_at": None}
       try:
           data = json.loads(pid_path.read_text(encoding="utf-8"))
           pid = int(data["pid"])
           started_at = str(data.get("started_at", ""))
       except (json.JSONDecodeError, KeyError, ValueError, OSError):
           return {"state": "idle", "pid": None, "started_at": None}
       try:
           os.kill(pid, 0)
           state = "running"
       except (ProcessLookupError, PermissionError):
           state = "stale"
       return {"state": state, "pid": pid, "started_at": started_at}
   ```

| os.kill(pid, 0) | PID from file used in signal call |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-17-02-01 | Tampering | .run-pid PID injection | mitigate | os.kill(pid, 0) is read-only liveness probe; wrap in try/except PermissionError + ProcessLookupError; coerce pid to int before use |
| T-17-02-02 | DoS | malformed .run-pid JSON | mitigate | json.JSONDecodeError caught; degrade to idle state (load_run_status never raises) |
| T-17-02-03 | Info Disclosure | PID + start time in dashboard tooltip | accept | dashboard is operator-only local tool; no PII; PID is process metadata |
| T-17-02-04 | Elevation | SIGINT handler removing files | accept | only removes .run-pid inside universe_dir which the runner owns; no privilege escalation |
