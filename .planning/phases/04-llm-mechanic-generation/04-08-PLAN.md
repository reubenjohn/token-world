Step A — Add helpers to `_helpers.py`:
```python
def _count_holds(ctx: "MechanicContext", actor: str) -> int:
    """Return the number of outgoing `holds` edges from actor."""
    return sum(1 for _ in ctx.neighbors(actor, relation="holds"))


def _refuse_with_narrative(ctx: "MechanicContext", actor: str, narrative: str) -> list["Mutation"]:
    """Emit a single mutation recording the refusal narrative on the actor.

    Shared blocked-by helper pattern (partial absorb of GAP-CROSS02 for Phase 4;
    full engine-level contract is Phase 5's responsibility).
    """
    return [ctx.mutate(actor, "last_refusal_narrative", narrative)]
```
