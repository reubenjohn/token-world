<objective>
Refactor the three LLM-calling classes (Classifier, Observer, ResidentAgent) to use the
`LLMBackend` abstraction delivered by Plan 01, in a fully backward-compatible way:

- Add `backend: LLMBackend | None = None` to each class's constructor.
- `__post_init__` (or `__init__` for ResidentAgent) implements wrap-or-default:
  - If `backend` is provided, use it.
  - Elif `client` is provided, wrap it in `AnthropicSDKBackend(self.client)`.
  - Else, call `get_backend()` (env-var dispatched factory).
- Replace direct `self.client.messages.create(...)` calls with `self.backend.call(...)`.
- Preserve all diagnostics, token-capture, and retry logic untouched.

exercises three layers:
1. **Backward compat:** existing `client=FakeClient(...)` patterns still work.

   identical results to the SDK path.

into the simulation pipeline.

### Classifier — AFTER (refactor target)

```python
from token_world.engine.llm_backend import (
    AnthropicSDKBackend,
    LLMBackend,
    get_backend,
)

@dataclass(slots=True)
class Classifier:
    client: Any = None  # deprecated; kept for backward compatibility (D-02)
    model: str = _MODEL
    max_tokens: int = 1024
    backend: LLMBackend | None = None

    def __post_init__(self) -> None:
        if self.backend is None:
            if self.client is not None:
                object.__setattr__(self, "backend", AnthropicSDKBackend(self.client))
            else:
                object.__setattr__(self, "backend", get_backend())

    # classify() unchanged

    def _send(self, user_prompt: str, *, tick_diag_ctx: Any, attempt: int = 1) -> str:
        # backend is always populated post __post_init__; mypy needs the assertion
        assert self.backend is not None
        raw = self.backend.call(
            model=self.model,
            system=_SYSTEM_PROMPT,
            prompt=user_prompt,
            max_tokens=self.max_tokens,
        )
        if tick_diag_ctx is not None:
            tick_diag_ctx.write_response(
                "classification",
                raw,
                suffix=f"_attempt{attempt}" if attempt > 1 else "",
            )
        return raw
```

### ResidentAgent — AFTER

```python
from token_world.engine.llm_backend import (
    AnthropicSDKBackend,
    LLMBackend,
    get_backend,
)

class ResidentAgent:
    def __init__(
        self,
        agent_id: str,
        session_id: str,
        personality: PersonalityBundle,
        memory: AgentMemory,
        client: Any = None,
        *,
        model: str = "claude-haiku-4-5",
        world_rules: str = "",
        backend: LLMBackend | None = None,
    ) -> None:
        self._agent_id = agent_id
        self._session_id = session_id
        self._personality = personality
        self._memory = memory
        self._client = client
        self._model = model
        self._world_rules = world_rules
        self._system_prompt = self._build_system_prompt()
        # Backend wrap-or-default (D-02, D-10)
        if backend is not None:
            self._backend = backend
        elif client is not None:
            self._backend = AnthropicSDKBackend(client)
        else:
            self._backend = get_backend()

    def run_turn(self) -> str:
        messages = self._build_messages()
        # Flatten messages into a single prompt string for LLMBackend.call()
        # interface. Format: "{role}: {content}" blocks separated by "\n\n".
        prompt = "\n\n".join(
            f"{m['role']}: {m['content']}" for m in messages
        )
        text = self._backend.call(
            model=self._model,
            system=self._system_prompt,
            prompt=prompt,
            max_tokens=256,
        )
        if not text or not text.strip():
            raise ValueError("LLM returned empty response in run_turn()")
        return text.strip()
```
