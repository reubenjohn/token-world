# Phase 19 Plan 01: Historical Tick Summary Migration

## Objective
Backfill pre-ENGINE-01 false-EXECUTED tick summaries in willowbrook so the quality scorer no longer counts them as ungrounded.

## Tasks

### Task 1: Create migration script
Create `scripts/migrate_tick_summaries.py` that:
- Scans all tick files for false-EXECUTED records (status=executed, refused=false, mutations.count=0, yielded=false, no mechanic_check_failed)
- `--dry-run`: prints table of affected ticks
- `--apply`: rewrites with refused=true, refusal_reason="mechanic_check_failed", status kept as-is but refused=true (scorer uses refused field)
- Idempotent

### Task 2: Create tests
Create `tests/test_scripts/test_migrate_tick_summaries.py` with synthetic tick fixtures.

### Task 3: Run migration on willowbrook
Execute dry-run and apply passes; verify idempotency.
