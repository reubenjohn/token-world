    - description = "Agent examines a nearby entity; records its visible properties."

    - check(): actor exists AND has located_in room. Return CheckResult(passed=True) if so.

    **pet.py** — PetMechanic
    - id = "pet"
    - description = "Agent pets a nearby animal; improves its mood."
    - voluntary = True, tags = ["social", "animal"]
    - check(): actor exists; there is at least one entity with subtype="animal" in actor's room.
      Return CheckResult(passed=False, reasons=["no animal nearby"]) if none found.
    - apply(): find the animal entity (first node with subtype="animal" in actor's room).
      Read current mood (string). Map mood transitions: "hostile"→"wary", "wary"→"neutral",
      "neutral"→"content", "content"→"purring", "purring"→"purring" (clamp at top).
      Write target.mood = new_mood. Return [mutation].

    **sharpen.py** — SharpenMechanic
    - id = "sharpen"
    - description = "Agent sharpens an item on a whetstone in the current room."
    - voluntary = True, tags = ["tool", "craft"]
    - check(): actor exists AND has located_in room AND room contains at least one entity with
      subtype="tool" and material="stone" (the whetstone). Also: actor must be carrying something
      (has outgoing "carrying" edge) or there is a "knife" entity in the room.
      Return CheckResult(passed=False, reasons=["no whetstone nearby"]) if whetstone absent.
    - apply(): find the whetstone. Find the carried item (prefer "carrying" neighbor; fall back
      to any entity in room with subtype="tool" that is NOT the whetstone).
      Read sharpness (float, default 0.5). Increment by 0.1, clamp to 1.0.
      Write target.sharpness = new_value. Return [mutation].

    **hum.py** — HumMechanic
    - id = "hum"
    - description = "Agent hums a tune; sets humming=True on the actor."
    - voluntary = True, tags = ["social", "expressive"]
    - check(): actor exists. Always passes (no preconditions beyond existence).
      Return CheckResult(passed=True).
    - apply(): return [ctx.mutate(ctx.actor, "humming", True)].

    **drop.py** — DropMechanic
    - id = "drop"
    - description = "Agent drops a carried item into the current room."
    - voluntary = True, tags = ["inventory"]
    - check(): actor exists AND has at least one outgoing "carrying" edge.
      Return CheckResult(passed=False, reasons=["not carrying anything"]) if no carried item.
    - apply(): find first "carrying" neighbor of actor.
      Get actor's located_in room.
      Remove edge (actor, item, "carrying").
      Add edge (room, item, "contains").
      Return [remove_edge_mutation, add_edge_mutation].
