**src/token_world/quality/scorer.py** — single-pass scanner:

```python
def score(universe_dir: Path, *, slug: str, last: int = 50) -> QualityReport:
    """Compute QualityReport for the last `last` ticks of the universe."""
```

Implementation approach:
1. Load tick files using `iter_tick_files(ticks_dir, since=last)` from `token_world.inspect._shared`. Handle missing dir gracefully (return INSUFFICIENT_DATA report).
2. Parse with `read_json_file`. Filter None. If 0 payloads, return INSUFFICIENT_DATA.
3. Single pass over payloads in chronological order, accumulating per-dimension accumulators.
4. After the pass, call each private scorer function to produce `DimensionResult`.
5. Call `compute_verdict` on the dimensions list.

Implement all 8 private dimension scorers:

**_score_groundedness(payloads):**
- Proxy: tick is "grounded" if NOT refused AND mutations.count > 0, OR refused (honest refusal), OR yielded.
- Ungrounded = executed (not refused, not yielded) AND mutations.count == 0.
- score = grounded_count / total; compare to GREEN/RED thresholds.
- detail = f"{grounded_count}/{total} grounded"

**_score_character_stability(payloads):**
- For each tick: `any(m in payload["action_text"].lower() for m in MARKERS)`.
- score = 1 - (breaks / total); detail = f"{breaks} breaks"

**_score_action_coherence(payloads):**
- Compute longest non-refuse streak: scan refused fields in order.
- Compute refuse_rate_10tick = (total_refuses / total) * 10.
- Both metrics must be in green for OK; either in red triggers FAIL; otherwise WARN.
- Return TWO numeric scores in detail: `streak={streak}  refuse_rate={rate:.1f}`
- Use streak as primary score field; status is the combined verdict.

**_score_refusal_cluster(payloads):**
- Scan in order, track current consecutive refuse run length; track max.
- score = max_consecutive; detail = f"max={max_consecutive}  (< 5)"
- OK if <= CLUSTER_WARN; WARN if < CLUSTER_RED; FAIL if >= CLUSTER_RED.

**_score_vocabulary_growth(payloads):**
- Accumulate set of seen mechanic IDs. On each new ID: increment novel count.
- Check stagnation: longest run without a new mechanic ID. If > VOCAB_STAGNANT_TICKS → RED.
- rate = novel_count / (len(payloads) / 10.0) — per 10-tick average.
- Green if VOCAB_RATE_MIN_GREEN <= rate <= VOCAB_RATE_MAX_GREEN and not stagnant.
- Red if rate > VOCAB_RATE_RED_HIGH or stagnant.
- detail = f"{novel_count} novel mechanics  rate={rate:.1f}/10t"

**_score_conservation_drift(payloads):**
- Count ticks where refused=True and refusal_reason and "conservation" in refusal_reason.lower().
- rollback_rate = conservation_count / total
- detail = f"{conservation_count}/{total} rollbacks"

**_score_graph_fanout(universe_dir, payloads):**
- Query `graph_snapshots` table from `universe_dir / "universe.db"`.
- Use the pattern from RESEARCH.md Pattern 4 (replicated above in interfaces).
- If DB missing or < 2 snapshots: return DimensionResult with status="OK", score=0.0, detail="insufficient history".
- Compute slope over up to 5 most-recent snapshots.
- Green if slope >= FANOUT_GREEN, red if slope <= FANOUT_RED, else warn.
- detail = f"slope={slope:.4f}/10t  (last {len(history)} snapshots)"

**_score_novel_subtype_rate(payloads):**
- For each tick's mutations.list, find entries where prop == "subtype" and the new value hasn't been seen before.
- Accumulate distinct new subtypes seen. rate = new_subtypes / (len(payloads) / 10.0).
- WARN if rate == 0 and len(payloads) >= 30; OK otherwise (no FAIL — dimension is informational per rubric gap).
- detail = f"{new_subtypes} new subtypes  rate={rate:.1f}/10t"

Command signature:
```python
@cli.command("quality")
@click.argument("slug")
@click.option(
    "--last",
    "last_n",
    type=int,
    default=50,
    show_default=True,
    help="Score the last N ticks (default 50, per rubric spec).",
)
@click.option(
    "--format",
    "fmt",
    type=click.Choice(["table", "json"]),
    default="table",
    show_default=True,
    help="Output format.",
)
def quality(slug: str, last_n: int, fmt: str) -> None:
    """Print a simulation quality scorecard across all 8 rubric dimensions.

    Reads tick_summaries/ and universe.db. No LLM calls. Thresholds from
    docs/quality/sim-quality-rubric.md. Exit 0 always (gate via
    scripts/check_quality_thresholds.py).
    """
    from token_world.quality import score
    from token_world.quality.report import render_json, render_table

    manager = UniverseManager()
    try:
        universe_dir = manager.load(slug)
    except FileNotFoundError as e:
        click.echo(f"Error: {e}", err=True)
        raise SystemExit(1) from e

    report = score(universe_dir, slug=slug, last=last_n)
    if fmt == "json":
        click.echo(render_json(report), nl=False)
    else:
        click.echo(render_table(report), nl=False)
```

Note: The CLI command exits 0 even on FAILED verdict — the CI gate (`scripts/check_quality_thresholds.py`, Plan 02) handles non-zero exits. Keeping the CLI exit at 0 makes `--format json` pipeable without special-casing.
