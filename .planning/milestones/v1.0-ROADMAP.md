## Overview

Token World v1.0 delivers a universe simulator where LLM-powered resident agents inhabit a text-based world and interact with an environment whose rules are procedurally authored on-the-fly. The core loop works end-to-end: the engine classifies agent actions, matches existing mechanics (or yields to the operator for mechanic authoring), executes them against the knowledge graph, and returns grounded observations synthesized from graph state. Attention/consciousness mechanics (sleep, daydream, autopilot travel, drunkenness) all compose through a single interruption-threshold pattern, validating the philosophy that emergent mechanics should derive from shared primitives.

This milestone establishes the full architecture — knowledge graph (NetworkX + SQLite, schema-less, snapshot-capable), mechanic framework (flat Python modules authored via normal SDLC, validated through a 6-stage gate), simulation engine (Haiku classifier + Sonnet observer + deterministic matcher with inversion-of-control yield), resident agent (personality-driven, memory-persistent, forkable sessions), and operator harness (Agent SDK driver that catches yield signals and spawns an Opus mechanic-authoring subagent). 35 hand-authored use cases drive regression testing. A pluggable LLM backend (AnthropicSDKBackend default, ClaudeCLIBackend for zero-cost live UAT via user's Claude subscription) enables live playtesting without API spend.
