The current Stages 2+3 in `run_tick` (after the classifier call, around line 337):

    match_result = self._matcher.match(verdict.classified, ...)

Replace with:

```python
# Non-VerdictOk path: classifier-level refusal — single decide() call, no iteration
if not isinstance(verdict, VerdictOk):
    decision = decide(verdict, None, action_text=action_text)
    # must be RefuseDecision
    return self._handle_refuse(
        decision=decision,
        verdict=verdict,
        tick_id_str=tick_id_str,
        tick_ctx=tick_ctx,
        start_time=start_time,
        classifier_in=classifier_in,
        classifier_out=classifier_out,
        action_text=action_text,
    )

# VerdictOk path: iterate sub-actions
all_match_results = []
sub_execute_decisions: list[tuple[ExecuteDecision, ClassifiedAction]] = []
all_classified_dicts: list[dict] = [ca.model_dump() for ca in verdict.actions]

for classified_action in verdict.actions:
    sub_match = self._matcher.match(classified_action, self._registry, self._graph)
    all_match_results.append(sub_match.model_dump())
    # Wrap as single-element VerdictOk for decide()
    sub_verdict = VerdictOk(actions=[classified_action], confidence=verdict.confidence)
    sub_decision = decide(sub_verdict, sub_match, action_text=action_text)

    if isinstance(sub_decision, YieldDecision):
        # First yield wins — halt immediately
        tick_ctx.write_matching(all_match_results)
        return self._handle_yield(
            decision=sub_decision,
            verdict=VerdictOk(actions=[classified_action], confidence=verdict.confidence),
            actor=actor,
            tick_id_str=tick_id_str,
            tick_ctx=tick_ctx,
            start_time=start_time,
            classifier_in=classifier_in,
            classifier_out=classifier_out,
            action_text=action_text,
        )
    elif isinstance(sub_decision, ExecuteDecision):
        sub_execute_decisions.append((sub_decision, classified_action))
    elif isinstance(sub_decision, RefuseDecision):
        # Record but continue — other sub-actions still run
        logger.debug(
            "Sub-action %r refused (%s); continuing composite tick",
            classified_action.verb,
            sub_decision.reason_code,
        )

tick_ctx.write_matching(all_match_results)

# If ALL sub-actions refused, treat the tick as refused
if not sub_execute_decisions:
    # Use the first sub-action's verdict for the refusal message
    first_ca = verdict.actions[0]
    sub_verdict_0 = VerdictOk(actions=[first_ca], confidence=verdict.confidence)
    first_match = self._matcher.match(first_ca, self._registry, self._graph)
    refuse_decision = decide(sub_verdict_0, first_match, action_text=action_text)
    return self._handle_refuse(
        decision=refuse_decision,  # type: ignore[arg-type]
        verdict=verdict,
        tick_id_str=tick_id_str,
        tick_ctx=tick_ctx,
        start_time=start_time,
        classifier_in=classifier_in,
        classifier_out=classifier_out,
        action_text=action_text,
    )

# Execute each matched sub-action; collect traces
# Use first execute decision as the "primary" for summary back-compat
primary_exec_decision, primary_classified = sub_execute_decisions[0]

return self._handle_execute_composite(
    decisions=sub_execute_decisions,
    verdict=verdict,
    classified_dicts=all_classified_dicts,
    actor=actor,
    tick_id_str=tick_id_str,
    next_tick=next_tick,
    pre_tick_snapshot_id=pre_tick_snapshot_id,
    tick_ctx=tick_ctx,
    start_time=start_time,
    classifier_in=classifier_in,
    classifier_out=classifier_out,
    action_text=action_text,
)
```

Add `_handle_execute_composite` method to `SimulationEngine`. This method:
1. Iterates `decisions` (list of `(ExecuteDecision, ClassifiedAction)`)
2. For each: constructs `MechanicContext` with the sub-action's `classified.target or actor`,
   calls `chain_engine.execute(mechanic, ctx)`, collects the trace
3. If any sub-action's `primary_trace.root.check_result.passed` is False, records it
   as a refuse but CONTINUES to next sub-action (check-failed sub-actions don't halt the tick)
4. After all sub-actions, combines all sub-traces into a single root trace:
   - Root trace node: use the first executed sub-action's mechanic_id; children include all
     other sub-action trace nodes as children of root (similar to `_trace_with_sweep` pattern)
5. Runs conservation check on combined mutations (same as current `_handle_execute`)
6. Runs passive sweep
7. Runs observer
8. Calls `_write_summary` passing `classified_actions=all_classified_dicts`
9. Returns `TickResult.ok`

Update `_write_summary` signature to add `classified_actions: list[dict] | None = None`
and pass it through to `build_tick_summary`.

Update all existing `_write_summary` call sites in engine.py to add `classified_actions=None`
(or omit — default covers it).
