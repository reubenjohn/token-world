## Goal

Thread a `selected_agent` reactive state through all four dashboard panels so the dashboard is multi-agent-ready without touching the engine.

## Context

**D-17 (unchanged):** Engine stays single-agent. Dashboard readiness is additive UI work only — no engine, graph, or mechanic changes.

**selected_agent contract:** `""` (empty string) means "All agents" and is the default when only one agent exists. Populated by a `ui.select()` dropdown above the tick stream. Shared between panels via closure state passed from `app.py`.

**Agent enumeration:** Uses `inspect/agents.py` pattern — `kg.nodes(type="agent")` via `KnowledgeGraph` loaded from `universe.db`. Refreshes on the same 2s cadence as the tick stream.

**Actor badge source:** `tick["classified_action"]["actor"]` if present, otherwise `""`. Added to the `build_card()` return dict as `"actor_id"`.

**Graph highlight:** `graph_canvas.py` already has `synthesise_property_edges` for `located_in` pseudo-edges (DASHBOARD-04). SC-2 adds a CSS outline to the selected agent node declaration in the Mermaid source, and highlights its `located_in` pseudo-edge by emitting it with a thicker arrow when it originates from the selected agent.

**Stats rollup:** `render_cells()` is a pure function returning `list[dict]` — extend it to accept optional `per_agent_yield` mapping and emit per-agent rows only when `agent_count > 1`.

**Test strategy:** No live NiceGUI server. Pure-function tests on `build_card`, `render_cells`, `synthesise_property_edges` shape changes. Integration tests use `fake_universe_with_graph` fixture (already has `alice` and `bob` as agent nodes).

```python
# Reactive selected_agent state (mutable dict as closure-safe ref)
selected_agent: dict[str, str] = {"value": ""}

# Agent selector — loads agent IDs from KG
from token_world.inspect.agents import aggregate as agents_aggregate

agent_state: dict[str, Any] = {"options": [""], "select_elem": None}

def _refresh_agent_options() -> None:
    try:
        report = agents_aggregate(universe_dir, slug=slug)
        ids = sorted(a.id for a in report.agents)
    except Exception:
        ids = []
    options = [""] + ids  # "" = All agents
    agent_state["options"] = options
    sel = agent_state.get("select_elem")
    if sel is not None:
        sel.options = options

with ui.column().classes("flex-1 min-w-[360px] max-w-[560px] gap-2"):
    ui.label("Tick stream").classes("text-lg font-semibold text-slate-200")

    # Agent selector (above tick stream)
    _refresh_agent_options()
    options = agent_state["options"]
    label_map = {"": "All agents"}  # display name for empty-string option

    def _on_agent_select(e: Any) -> None:
        selected_agent["value"] = e.value if e.value else ""

    sel_elem = ui.select(
        options,
        value="",
        label="Agent",
        on_change=_on_agent_select,
    ).classes("w-full bg-slate-800 text-slate-100").props("dense dark")
    agent_state["select_elem"] = sel_elem
    ui.timer(2.0, _refresh_agent_options)

    mount_tick_stream_panel(universe_dir, slug, selected_agent=selected_agent)
```

Move the tick stream column mount from `_mount_main_body` into the new structure above — it replaces the existing `with ui.column()...` block for the tick stream.

In the node-declaration loop, after emitting the base node line, append a Mermaid `style` override for the selected node:

For the pseudo-edge loop, highlight `located_in` edges from the selected agent with a thicker dashed arrow. Since Mermaid does not support per-edge styles in `flowchart LR` without `linkStyle` (which requires a numeric index), use a simpler approach: emit the selected agent's `located_in` pseudo-edge using a solid highlighted arrow instead of dashed, to make it visually distinct:
```python
if selected_agent_id and pseudo["src"] == selected_agent_id and pseudo.get("relation") == "located_in":
    # Highlighted: solid amber arrow for selected agent's location
    lines.append(f"    {src} == {relation} ==> {dst}")
else:
    # Standard dashed pseudo-edge
    if relation:
        lines.append(f"    {src} -.{relation}.-> {dst}")
    else:
        lines.append(f"    {src} -.-> {dst}")
```

```python
from __future__ import annotations

from token_world.dashboard.panels.stats import render_cells, StatsReport


def test_render_cells_no_per_agent_rollup_for_single_agent() -> None:
    report = StatsReport(slug="test", tick_count=5, yield_count=1)
    per_agent = {"alice": {"ticks": 5, "yields": 1}}
    cells = render_cells(report, per_agent_yield=per_agent)
    labels = [c["label"] for c in cells]
    # Single actor -> no per-agent breakdown rows
    assert not any(l.startswith("Yield (") for l in labels)


def test_render_cells_per_agent_rollup_shown_for_two_agents() -> None:
    report = StatsReport(slug="test", tick_count=10, yield_count=3)
    per_agent = {
        "alice": {"ticks": 6, "yields": 2},
        "bob": {"ticks": 4, "yields": 1},
    }
    cells = render_cells(report, per_agent_yield=per_agent)
    labels = [c["label"] for c in cells]
    assert "Yield (alice)" in labels
    assert "Yield (bob)" in labels
    alice_cell = next(c for c in cells if c["label"] == "Yield (alice)")
    assert alice_cell["value"] == "33.3%"


def test_render_cells_per_agent_rollup_hidden_when_none() -> None:
    report = StatsReport(slug="test")
    cells = render_cells(report, per_agent_yield=None)
    labels = [c["label"] for c in cells]
    assert not any(l.startswith("Yield (") for l in labels)
```
