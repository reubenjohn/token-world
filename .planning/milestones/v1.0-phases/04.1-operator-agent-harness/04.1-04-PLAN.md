<objective>
Surface the operator harness through developer-friendly CLI commands. The human operator runs `token-world run-tick`, sees a yield, and either lets the harness auto-author the mechanic (default) or opts out with `--manual` to author by hand. `inspect-yield` and `replay-tick` make the operator diagnostics navigable without crack-opening JSON by hand. `resume-tick` is the escape hatch for manually-authored flows.

Purpose: D-05 requires dev-UX parity with a human-driven flow; without CLI commands, the programmatic harness is only reachable via Python. This plan turns "Agent SDK driver" into "invoke `token-world run-tick` and go make coffee." It also implements D-16: all diagnostics rendering goes through the reader API — no regex parsing anywhere.

Output: 4 new commands on the existing `token-world` Click group; a shared `cli_support` module with universe resolution + renderer functions; test coverage via Click's CliRunner (unit-level; integration smoke is covered by 04.1-03).
</objective>

Click commands:

```
token-world run-tick [UNIVERSE_SLUG] [--manual] [--format json|human] [--tick TICK_ID]
token-world inspect-yield [UNIVERSE_SLUG] [--tick TICK_ID] [--format human|json]
token-world resume-tick [UNIVERSE_SLUG] --tick TICK_ID
token-world replay-tick [UNIVERSE_SLUG] TICK_ID [--format human|json]
```

Command semantics:

- **run-tick:** If this is the first Phase 4.1 release (no Phase 5 engine yet), `run-tick` without `--tick` is *not fully functional* because there's no real engine to call. Two options:
  1. (**Pick this one**) `run-tick` expects a YieldSignal to exist already — useful for drain/resume workflows. Without `--tick` and with no halted tick, exits 0 with message `"No halted ticks. Start a real simulation tick by [Phase 5 TBD]."`. Document clearly in `--help`.
  2. Fallback: `run-tick --stub verb=<verb> --stub actor=<actor>` manually fabricates a yield via EngineStub (useful during 04.1 dogfooding).
  Actually — do BOTH: honour a `--stub <KEY=VALUE>...` option (repeatable) for explicit stub invocation, AND auto-detect the latest halted tick when `--stub` is absent. This preserves the "run-tick drains any halted tick" semantic Phase 5 will rely on.
- **inspect-yield:** Reads `<universe>/diagnostics/tick_<id>/operator/yield_signal.json` via OperatorDiagnosticsReader; --tick defaults to latest halted.
- **resume-tick:** Thin wrapper; uses the Agent SDK briefly to call `mcp__token-world__resume_tick` via a one-shot non-interactive session (or more directly: invoke the MCP server's method via subprocess using the existing `token-world-mcp` CLI). Pick the simpler path — a short SDK session is fine, and preserves D-05 parity.
- **replay-tick:** `OperatorDiagnosticsReader(universe, tick_id)` → render. --format human uses colored click.echo; --format json emits `{"yield_signal": ..., "attempts": [...], "validation_reports": [...], "mechanic_diff": "...", "resume_outcome": ...}`.

Exit codes:
- `run-tick`: 0 on success, 1 on harness failure, 2 on universe-not-found, 3 on no-halted-tick-no-stub.
- `inspect-yield`: 0 if found, 2 if universe missing, 4 if no yield for specified tick.
- `resume-tick`: 0 on success, 1 on MCP failure, 2 on universe missing.
- `replay-tick`: 0 if found, 2 if universe missing, 4 if tick not found.

    ```python
    from __future__ import annotations

    import json
    import os
    from pathlib import Path
    from typing import TYPE_CHECKING

    import click

    from token_world.operator.diagnostics import OperatorDiagnosticsReader
    from token_world.operator.yield_signal import YieldSignal

    if TYPE_CHECKING:
        from token_world.universe.manager import UniverseManager


    _UNIVERSE_MARKERS = (".mcp.json",)  # mechanics/ checked as dir


    def resolve_universe(slug_or_none: str | None) -> Path:
        if slug_or_none:
            from token_world.universe.manager import UniverseManager
            return UniverseManager().load(slug_or_none)  # check actual signature
        env = os.environ.get("TOKEN_WORLD_UNIVERSE")
        if env:
            return Path(env)
        cwd = Path.cwd()
        if all((cwd / m).exists() for m in _UNIVERSE_MARKERS) and (cwd / "mechanics").is_dir():
            return cwd
        raise click.ClickException(
            "No universe specified. Options:\n"
            "  (1) pass a slug: `token-world <cmd> <slug>`\n"
            "  (2) set env var: `TOKEN_WORLD_UNIVERSE=/path/to/universe`\n"
            "  (3) run inside a universe folder (one containing .mcp.json and mechanics/)"
        )


    def render_yield_human(signal: YieldSignal) -> str:
        lines = [
            f"Tick {signal.tick_id} — yielded ({signal.reason})",
            f"Universe: {signal.universe_path}",
            f"Action text: {signal.action_text!r}",
            "",
            "Classified action:",
            f"  verb:   {signal.classified_action.get('verb')}",
            f"  actor:  {signal.classified_action.get('actor')}",
            f"  target: {signal.classified_action.get('target')}",
            f"  params: {json.dumps(signal.classified_action.get('params', {}), indent=2)}",
            "",
            f"Actor state: {json.dumps(signal.actor_state, indent=2)}",
            f"Candidate mechanics: {signal.candidate_mechanic_ids or '(none)'}",
        ]
        return "\n".join(lines)


    def render_yield_json(signal: YieldSignal) -> str:
        return signal.to_json()


    def render_replay_human(reader: OperatorDiagnosticsReader) -> str:
        try:
            signal = reader.yield_signal()
        except FileNotFoundError:
            return f"No operator session found for tick {reader.tick_id}."
        attempts = reader.attempts()
        validations = reader.validation_reports()
        diff = reader.mechanic_diff()
        outcome = reader.resume_outcome()
        out = [
            render_yield_human(signal),
            "",
            f"Authoring attempts: {len(attempts)} message(s)",
            f"Validation reports: {len(validations)}",
        ]
        for i, rep in enumerate(validations, 1):
            passed = "PASS" if rep.get("passed") else "FAIL"
            out.append(f"  attempt_{i:02d}: {passed} ({len(rep.get('findings', []))} findings)")
        out.append("")
        if diff is not None:
            out.append(f"Mechanic diff ({len(diff)} chars):")
            out.append(diff[:500] + ("..." if len(diff) > 500 else ""))
        else:
            out.append("Mechanic diff: (none written)")
        if outcome is None:
            out.append("\nSession not closed (resume_outcome.json missing).")
        else:
            out.append(
                f"\nOutcome: success={outcome.get('success')} "
                f"mechanic_id={outcome.get('mechanic_id')!r} "
                f"cost_usd={outcome.get('cost_usd')} turns={outcome.get('turns')}"
            )
            if outcome.get("error"):
                out.append(f"Error: {outcome['error']}")
        return "\n".join(out)


    def render_replay_json(reader: OperatorDiagnosticsReader) -> str:
        try:
            signal_json = reader.yield_signal().to_json()
        except FileNotFoundError:
            return json.dumps({"error": "no_session", "tick_id": reader.tick_id}, indent=2)
        payload = {
            "yield_signal": json.loads(signal_json),
            "attempts": reader.attempts(),
            "validation_reports": reader.validation_reports(),
            "mechanic_diff": reader.mechanic_diff(),
            "resume_outcome": reader.resume_outcome(),
        }
        return json.dumps(payload, indent=2, sort_keys=True)


    def latest_halted_tick(universe: Path) -> str | None:
        diag_dir = universe / "diagnostics"
        if not diag_dir.is_dir():
            return None
        halted = []
        for tick_dir in diag_dir.iterdir():
            if not tick_dir.is_dir() or not tick_dir.name.startswith("tick_"):
                continue
            op_dir = tick_dir / "operator"
            if not op_dir.is_dir():
                continue
            outcome_file = op_dir / "resume_outcome.json"
            if not outcome_file.exists():
                halted.append(tick_dir.name)
                continue
            try:
                outcome = json.loads(outcome_file.read_text(encoding="utf-8"))
                if not outcome.get("success", False):
                    halted.append(tick_dir.name)
            except json.JSONDecodeError:
                halted.append(tick_dir.name)  # corrupted == needs investigation == "halted"
        if not halted:
            return None
        # Numerical sort on tick id suffix; fall back to lexicographic for non-numeric
        def sort_key(name: str) -> tuple[int, str]:
            suffix = name.removeprefix("tick_")
            try:
                return (int(suffix), name)
            except ValueError:
                return (-1, name)
        halted.sort(key=sort_key)
        # Strip the "tick_" prefix to return the id string
        return halted[-1].removeprefix("tick_")
    ```

    ```python
    @cli.command("run-tick")
    @click.argument("universe_slug", required=False)
    @click.option("--manual", is_flag=True,
                  help="Print yield and exit; do not invoke mechanic-author subagent.")
    @click.option("--tick", "tick_id", default=None,
                  help="Specific halted tick; defaults to latest.")
    @click.option("--format", "output_format",
                  type=click.Choice(["human", "json"]), default="json",
                  help="Output format (default json for programmatic callers).")
    @click.option("--stub", "stub_kvs", multiple=True,
                  help="Fabricate a yield via EngineStub. Repeatable KEY=VALUE (verb=..., actor=...).")
    def run_tick(universe_slug, manual, tick_id, output_format, stub_kvs):
        universe = _cli_resolve_or_exit(universe_slug, exit_code=2)
        # Build or load the yield signal.
        # BLOCKER-3: when --stub fabricates a signal, it MUST be persisted
        # to the operator diagnostics namespace BEFORE we decide about
        # --manual. Otherwise inspect-yield / replay-tick can't see it,
        # which breaks Plan 05's smoke test ("run run-tick --stub --manual
        # then inspect-yield must show the signal"). The --manual flag
        # controls whether the subagent spawns, NOT whether the signal
        # is written.
        if stub_kvs:
            effective_tick = tick_id or "tick_1"
            signal = _build_stub_signal(universe, stub_kvs, effective_tick)
            # Persist the fabricated signal via the Phase-4 sink so every
            # downstream operator CLI (inspect-yield, replay-tick, resume-tick)
            # finds it on disk. Open the session and immediately close it
            # with a "pending" outcome so the tick shows up as halted until
            # the harness (or a manual resume) updates resume_outcome.
            from token_world.mechanic.diagnostics import DiagnosticsSink
            sink = DiagnosticsSink(universe)
            with sink.open_operator_session(effective_tick) as ctx:
                ctx.write_yield_signal(signal)
                if manual:
                    ctx.close({
                        "success": False,
                        "mechanic_id": None,
                        "cost_usd": None,
                        "turns": 0,
                        "tick_continued": False,
                        "error": "manual_mode_no_harness_invoked",
                    })
            # Re-read tick_id for downstream messaging
            tick_id = effective_tick
        else:
            tick_id = tick_id or latest_halted_tick(universe)
            if tick_id is None:
                click.echo("No halted ticks. Use --stub to fabricate one, or wait for Phase 5 engine.", err=True)
                sys.exit(3)
            signal = OperatorDiagnosticsReader(universe, tick_id).yield_signal()

        if manual:
            # Stub path has already persisted+closed above; non-stub --manual
            # doesn't rewrite the existing session (signal was loaded from it).
            click.echo(render_yield_json(signal) if output_format == "json" else render_yield_human(signal))
            return  # exit 0

        try:
            harness = OperatorHarness(universe)
            result = asyncio.run(harness.handle_yield(signal))
        except Exception as e:
            click.echo(f"Harness failed: {e}", err=True)
            sys.exit(1)

        payload = {
            "success": result.success,
            "tick_id": result.tick_id,
            "mechanic_id": result.mechanic_id,
            "attempts": result.attempts,
            "cost_usd": result.cost_usd,
            "turns": result.turns,
            "error": result.error,
        }
        if output_format == "json":
            click.echo(json.dumps(payload, indent=2))
        else:
            click.echo(f"{'SUCCESS' if result.success else 'FAILURE'} tick={result.tick_id} mechanic={result.mechanic_id}")
        if not result.success:
            sys.exit(1)


    def _build_stub_signal(universe: Path, stub_kvs: tuple[str, ...], tick_id: str) -> YieldSignal:
        kwargs: dict[str, object] = {}
        for kv in stub_kvs:
            if "=" not in kv:
                raise click.ClickException(f"--stub expects KEY=VALUE; got {kv!r}")
            key, value = kv.split("=", 1)
            kwargs[key] = value
        if "verb" not in kwargs or "actor" not in kwargs:
            raise click.ClickException("--stub requires at least verb=... and actor=...")
        return EngineStub(universe).fabricate_yield(tick_id=tick_id, **kwargs)  # type: ignore[arg-type]


    def _cli_resolve_or_exit(slug: str | None, *, exit_code: int) -> Path:
        try:
            return resolve_universe(slug)
        except click.ClickException as e:
            click.echo(f"Error: {e.message}", err=True)
            sys.exit(exit_code)
    ```
