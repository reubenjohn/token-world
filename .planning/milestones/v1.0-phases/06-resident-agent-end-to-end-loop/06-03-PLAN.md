<objective>
Operationalize DVAL-03 (and contribute to TEST-04) by turning the 35 Phase-3 use-case
manifests into an executable regression test suite. Each UC becomes one parametrized test
case that exercises the real engine pipeline (classifier → matcher → mechanics →
conservation → observer → tick summary) using fake LLM clients, then asserts the final graph
state against the manifest's declared graph_assertions and the tick outcome against its
expected_outcome.

This is independent of 06-01 (resident) and 06-02 (compressor) — it only touches
`tests/test_regression/`. Safe to run in parallel (Wave 1).

  UC-V01..UC-V07, UC-E01..UC-E06)
- `@pytest.mark.regression` marker registered in pyproject.toml (add if missing)

       """Regression suite fixtures: fake LLM clients + graph-builder exec."""
       from __future__ import annotations
       from pathlib import Path
       from typing import Any
       import pytest
       from token_world.engine import SimulationEngine
       from token_world.engine.models import ClassifiedAction, VerdictOk
       from token_world.graph import KnowledgeGraph

       _EXEC_NS_BASE: dict[str, Any] = {
           "__builtins__": {
               "range": range, "print": print, "len": len,
               "list": list, "dict": dict, "set": set, "tuple": tuple,
               "str": str, "int": int, "float": float, "bool": bool,
               "True": True, "False": False, "None": None,
           }
       }

       def exec_graph_builder(code: str, kg: KnowledgeGraph) -> None:
           """Execute a UC manifest's setup.graph_builder code against kg.

           Namespace is restricted to prevent __import__ and other escapes.
           Mirrors tests/test_integration/ pattern.
           """
           ns = {**_EXEC_NS_BASE, "kg": kg}
           exec(compile(code, "<uc_setup>", "exec"), ns, ns)  # noqa: S102

       class FakeClassifier:
           last_input_tokens = 0
           last_output_tokens = 0
           def __init__(self, classified_dict: dict[str, Any]):
               self._classified_dict = classified_dict
           def classify(self, action_text, actor, *, available_verbs, known_node_ids,
                        min_confidence, tick_diag_ctx=None):
               return VerdictOk(
                   classified=ClassifiedAction(**{"actor": actor, **self._classified_dict}),
                   confidence=0.99,
               )

       class FakeObserver:
           last_input_tokens = 0
           last_output_tokens = 0
           def synthesize(self, *, projection, trace, refusal_narrative, actor_id,
                          action_text, tick_diag_ctx=None) -> str:
               if refusal_narrative is not None:
                   return refusal_narrative
               return "Action succeeded."

       @pytest.fixture
       def fake_anthropic_client():
           # Bare mock is sufficient because FakeClassifier/FakeObserver bypass the client.
           import unittest.mock as mock
           return mock.MagicMock()

       @pytest.fixture
       def build_engine(tmp_path, fake_anthropic_client):
           """Return a callable (kg, classified_dict) -> SimulationEngine."""
           def _build(kg: KnowledgeGraph, classified_dict: dict[str, Any]) -> SimulationEngine:
               universe = tmp_path / "test_universe"
               universe.mkdir(parents=True, exist_ok=True)
               # Engine expects universe.yaml/mechanics/; scaffold minimally.
               (universe / "mechanics").mkdir(exist_ok=True)
               engine = SimulationEngine(universe, graph=kg, anthropic_client=fake_anthropic_client)
               # Swap in fakes (attribute-level patch; acceptable in test infra)
               engine._classifier = FakeClassifier(classified_dict)
               engine._observer = FakeObserver()
               return engine
           return _build
       ```

       """Regression suite: 35 Phase-3 UC manifests run through the real engine pipeline.

       DVAL-03. Run with: `uv run pytest -m regression`.
       """
       from __future__ import annotations
       from pathlib import Path
       import pytest
       from token_world.graph import KnowledgeGraph
       from token_world.use_cases.loader import load_use_case
       from tests.test_regression.conftest import exec_graph_builder

       _UC_ROOT = Path(__file__).resolve().parents[2] / ".planning" / "use-cases"
       _UC_PATHS = sorted(p for p in _UC_ROOT.rglob("UC-*.md") if p.is_file())

       pytestmark = pytest.mark.regression

       @pytest.mark.parametrize("manifest_path", _UC_PATHS, ids=lambda p: p.stem)
       def test_use_case_regression(manifest_path: Path, build_engine, tmp_path):
           fm, _body = load_use_case(manifest_path)
           if not fm.get("actions"):
               pytest.skip(f"{fm.get('id')}: no actions declared")
           first_action = fm["actions"][0]
           if "classified" not in first_action:
               pytest.skip(f"{fm.get('id')}: missing classified block")

           # Build graph and execute setup
           kg_db_path = tmp_path / "regression_universe.db"
           kg = KnowledgeGraph(db_path=kg_db_path)
           graph_builder = fm["setup"]["graph_builder"]
           exec_graph_builder(graph_builder, kg)
           kg.save()

           engine = build_engine(kg, first_action["classified"])
           actor = first_action["actor"]
           intent = first_action["intent"]
           result = engine.run_tick(intent, actor=actor)

           expected_outcome = fm.get("expected_outcome", "pass")
           kind_map = {"pass": "ok", "yield": "yielded", "blocked": "refused"}
           expected_kind = kind_map[expected_outcome]
           assert result.kind == expected_kind, (
               f"{fm['id']}: expected_outcome={expected_outcome} → expected kind={expected_kind}, "
               f"got kind={result.kind} refusal_reason={result.refusal_reason}"
           )

           # Graph assertions (only for pass outcomes; yield/blocked paths don't mutate graph)
           if expected_kind != "ok":
               return
           observations = fm.get("expected_observations", [])
           if not observations:
               return
           for assertion in observations[0].get("graph_assertions", []):
               _verify_assertion(assertion, kg, fm["id"])

       def _verify_assertion(assertion: dict, kg: KnowledgeGraph, uc_id: str) -> None:
           kind = assertion["kind"]
           if kind == "has_node":
               assert kg.has_node(assertion["node"]), f"{uc_id}: missing node {assertion['node']}"
           elif kind == "has_edge":
               assert kg.has_edge(assertion["src"], assertion["dst"]), (
                   f"{uc_id}: missing edge {assertion['src']}→{assertion['dst']}"
               )
           elif kind == "has_property":
               node = assertion["node"]
               prop = assertion["property"]
               assert prop in kg.query_node(node), f"{uc_id}: {node} missing property {prop}"
           elif kind == "property_equals":
               node = assertion["node"]
               prop = assertion["property"]
               expected = assertion["value"]
               actual = kg.query_node(node).get(prop)
               assert actual == expected, f"{uc_id}: {node}.{prop} expected {expected}, got {actual}"
           elif kind == "not_has_edge":
               assert not kg.has_edge(assertion["src"], assertion["dst"]), (
                   f"{uc_id}: unexpected edge {assertion['src']}→{assertion['dst']}"
               )
           elif kind == "not_has_property":
               node = assertion["node"]
               prop = assertion["property"]
               assert prop not in kg.query_node(node), (
                   f"{uc_id}: {node} unexpectedly has property {prop}"
               )
           else:
               pytest.fail(f"{uc_id}: unsupported assertion kind {kind!r}")
       ```

<risks>
- **UC manifests without `classified` blocks**: some older UCs may not have this. The test
  skips them with a reason. That's acceptable — skipped tests are visible and actionable.
  Do NOT fail on missing classified; UC authors can add it later.
- **exec() security**: we restrict `__builtins__`. UC graph_builder code is git-committed by
  project maintainers (trusted); this is defence-in-depth per 06-RESEARCH Pitfall 2.
- **Test isolation**: each UC gets its own tmp_path → isolated DB, isolated universe dir. No
  cross-test state leak.
- **SimulationEngine scans mechanics/ at the top of every run_tick** (D-02). The test's
  universe dir has an empty `mechanics/` — so real mechanics are not available. The engine
  will yield for any classified action that has no matching mechanic. UCs with
  expected_outcome="pass" will FAIL unless we also bring in the seed mechanic library.
- **Seed mechanics for regression**: to exercise REAL mechanic matching, the engine needs
  access to the flat seed library. Two options:
  (a) Copy seed mechanics into the tmp universe (slow, complex).
  (b) Point the engine at `src/token_world/mechanic/seeds/` as the mechanics dir.
  For v1, use (b): in `build_engine`, symlink or copy-on-write the seeds dir to
  `universe/mechanics/`. Add to conftest build_engine:
  ```python
  import shutil
  seeds_src = Path("src/token_world/mechanic/seeds")
  if seeds_src.exists():
      for py_file in seeds_src.glob("*.py"):
          if py_file.name.startswith("_") or py_file.name == "__init__.py":
              continue
          shutil.copy2(py_file, universe / "mechanics" / py_file.name)
  ```
  This brings in MECH01-MECH27 for the regression suite to exercise.
- **Expected failures are informative**: the project currently lists UC gaps in each
  manifest's `gaps:` block. Failures in the suite correspond to those gaps. Record pass/fail
  count in 06-03-SUMMARY.md as the regression's baseline — future phases that close gaps
  will improve the number.
- **File overlap**: none with 06-01 or 06-02; 06-03 only modifies tests/ + pyproject.toml.
  pyproject.toml change is in [tool.pytest.ini_options.markers] — non-conflicting with any
  other Wave 1 plan.
- **CI runtime cost**: 35 UC runs × (classifier→match→execute→observe) ≈ <1s each with
  fake LLMs. Total ~30s. Acceptable for a regression gate.
</risks>
