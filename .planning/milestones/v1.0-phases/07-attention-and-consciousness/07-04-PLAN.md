<objective>
Wire the Phase 7 long-running infrastructure into the live engine pipeline. This plan brings
together:
- Plan 01's `LongRunningAction`, `ThresholdSpec`, `ThresholdEvaluator` dataclasses
- Plan 02's `VisibilityProjector(actor_id, attention_state=...)` extension
- Plan 03's `MechanicContext.begin_long_action()` helper

It introduces:
- `LongRunningHook` — the per-tick orchestrator that advances turns_elapsed, evaluates
  thresholds, and emits the right observation (D-06, D-08, D-09, D-10, D-15, D-20, D-22)
- The engine's top-of-`run_tick` LRA detection and synthetic continuation routing (D-07, D-11)
- The tick summary `long_running_action` field (D-17)
- The PlaytestRunner integration that suppresses `agent.run_turn()` calls during continuations
  (Pitfall 2 + D-07 transparent runner support)

Purpose: This is the keystone plan — it makes the abstract pattern from Plans 01-03 actually
do something. After this plan, an LRA written to a graph node by a mechanic will:
1. Survive across tick boundaries via graph persistence (D-02, free)
2. Cause subsequent ticks to skip the classifier/matcher/decider (D-07)
3. Get its turns_elapsed advanced and thresholds evaluated each tick (D-06, D-08, D-09)
4. Modulate perception via attention_state (D-12, free via Plan 02)
5. Produce a grounded "time passes" or interruption observation (D-10, D-22)
6. Show up in the tick summary so diagnostics can audit (D-17)
7. Be cancellable by a real agent action (D-11)

Output: New `engine/long_running_hook.py`, modifications to `engine.py`, `models.py`,
`summary_writer.py`, `playtest/runner.py`, plus three new test modules. Plans 05-07 (seed
mechanics) consume this plumbing in Wave 4 and add no further engine code.

    ```python
    """LongRunningHook (D-06, D-15, D-20) — per-tick orchestration of long-running actions.

    Reads current_long_action from the actor's graph node, advances turns_elapsed,
    evaluates thresholds, and emits an interruption / completion / time-passes
    observation. Returns a HookResult that the engine uses to build the tick
    summary and the TickResult.

    Pitfall 1 mitigation: this hook is invoked ONLY on the synthetic continuation
    path (engine._handle_long_running_tick). It is NOT invoked on _handle_execute
    (where a mechanic just called begin_long_action). This means the FIRST hook
    call for any LRA happens on the tick AFTER LRA-start, with turns_elapsed=0
    on the graph; the hook advances to 1 and evaluates. An LRA that started in
    a noisy room therefore does not insta-cancel.
    """
    from __future__ import annotations

    import logging
    from dataclasses import dataclass
    from typing import Any

    from token_world.engine.long_running import ThresholdEvaluator, ThresholdSpec

    logger = logging.getLogger(__name__)

    _LRA_PROPERTY = "current_long_action"
    _TIME_PASSES_TEMPLATE = "Time passes. You continue {action_text}."

    @dataclass(frozen=True, slots=True)
    class HookResult:
        active: bool                         # was an LRA running?
        interrupted: bool                    # did a threshold fire?
        completed: bool                      # did turns_elapsed reach turns_total?
        continuing: bool                     # action still running, no interruption
        fired_threshold: ThresholdSpec | None
        observation: str | None              # narrative for tick (interruption / completion / time passes)
        attention_state: dict | None         # echoed from lra.payload for engine to pass to projector
        action_text: str                     # echoed from lra for tick summary

        @classmethod
        def inactive(cls) -> "HookResult":
            return cls(False, False, False, False, None, None, None, "")

    class LongRunningHook:
        """Per-tick orchestration of an actor's long-running action (D-06)."""

        def process(
            self,
            *,
            actor: str,
            projection: dict,
            graph: Any,           # KnowledgeGraph (avoid import cycle in type)
            tick_id_str: str,
            observer: Any,        # Observer (avoid import cycle)
            tick_diag_ctx: Any,   # Phase 4 TickDiagnostics ctx for observer
        ) -> HookResult:
            # Step 1: Pitfall 3 — graceful no-op when actor missing or no LRA
            if not graph.has_node(actor):
                return HookResult.inactive()
            try:
                lra = graph.query(actor, _LRA_PROPERTY)
            except KeyError:
                return HookResult.inactive()
            if lra is None or not isinstance(lra, dict):
                return HookResult.inactive()

            action_text = lra.get("action_text", "")
            payload = lra.get("payload", {}) if isinstance(lra.get("payload"), dict) else {}
            attention_state = payload.get("attention_state")
            if not isinstance(attention_state, dict):
                attention_state = None

            # Step 2: advance turns_elapsed FIRST (Pitfall 1 — first hook call sees 1, not 0)
            new_elapsed = int(lra.get("turns_elapsed", 0)) + 1
            updated = dict(lra)
            updated["turns_elapsed"] = new_elapsed
            graph.set(actor, _LRA_PROPERTY, updated)

            # Step 3: evaluate thresholds
            thresholds = lra.get("thresholds", []) or []
            fired = ThresholdEvaluator.evaluate(thresholds, projection)
            if fired is not None:
                # Interruption path (D-10): clear LRA and synthesise grounded narrative
                graph.set(actor, _LRA_PROPERTY, None)
                observation = self._synthesise_interruption(
                    observer=observer,
                    projection=projection,
                    actor=actor,
                    action_text=action_text,
                    fired=fired,
                    tick_diag_ctx=tick_diag_ctx,
                )
                return HookResult(
                    active=True,
                    interrupted=True,
                    completed=False,
                    continuing=False,
                    fired_threshold=fired,
                    observation=observation,
                    attention_state=attention_state,
                    action_text=action_text,
                )

            # Step 4: completion check
            turns_total = lra.get("turns_total")
            if turns_total is not None and new_elapsed >= int(turns_total):
                graph.set(actor, _LRA_PROPERTY, None)
                observation = self._synthesise_completion(
                    observer=observer,
                    projection=projection,
                    actor=actor,
                    action_text=action_text,
                    tick_diag_ctx=tick_diag_ctx,
                )
                return HookResult(
                    active=True,
                    interrupted=False,
                    completed=True,
                    continuing=False,
                    fired_threshold=None,
                    observation=observation,
                    attention_state=attention_state,
                    action_text=action_text,
                )

            # Step 5: continuing — static template (D-22)
            return HookResult(
                active=True,
                interrupted=False,
                completed=False,
                continuing=True,
                fired_threshold=None,
                observation=_TIME_PASSES_TEMPLATE.format(action_text=action_text),
                attention_state=attention_state,
                action_text=action_text,
            )

        def _synthesise_interruption(
            self,
            *,
            observer: Any,
            projection: dict,
            actor: str,
            action_text: str,
            fired: ThresholdSpec,
            tick_diag_ctx: Any,
        ) -> str:
            from token_world.mechanic.trace import ExecutionTrace, TraceNode
            from token_world.mechanic.protocol import CheckResult
            # Minimal trace for the synthetic continuation tick — no real mechanic ran
            stub_node = TraceNode(
                mechanic_id="continue_long_action",
                actor=actor,
                target=actor,
                check_result=CheckResult(passed=True),
                mutations=[],
            )
            trace = ExecutionTrace(root=stub_node, total_mechanics_executed=0, max_depth_reached=0)
            interruption_context = {
                "interrupted_by": {
                    "property": fired.property,
                    "op": fired.op,
                    "value": fired.value,
                },
                "long_action": action_text,
            }
            return observer.synthesize(
                projection=projection,
                trace=trace,
                refusal_narrative=None,
                actor_id=actor,
                action_text=f"[interrupted: {action_text}]",
                tick_diag_ctx=tick_diag_ctx,
                interruption_context=interruption_context,
            )

        def _synthesise_completion(
            self,
            *,
            observer: Any,
            projection: dict,
            actor: str,
            action_text: str,
            tick_diag_ctx: Any,
        ) -> str:
            from token_world.mechanic.trace import ExecutionTrace, TraceNode
            from token_world.mechanic.protocol import CheckResult
            stub_node = TraceNode(
                mechanic_id="continue_long_action",
                actor=actor,
                target=actor,
                check_result=CheckResult(passed=True),
                mutations=[],
            )
            trace = ExecutionTrace(root=stub_node, total_mechanics_executed=0, max_depth_reached=0)
            interruption_context = {"completed": True, "long_action": action_text}
            return observer.synthesize(
                projection=projection,
                trace=trace,
                refusal_narrative=None,
                actor_id=actor,
                action_text=f"[completed: {action_text}]",
                tick_diag_ctx=tick_diag_ctx,
                interruption_context=interruption_context,
            )
    ```

        self,
        *,
        actor: str,
        tick_id_str: str,

        start_time: float,
    ) -> TickResult:
        """D-07: synthetic continuation tick for an active long-running action.

        Skips classifier/matcher/decider; calls the hook to advance turns_elapsed and
        evaluate thresholds; runs passive sweep; builds tick summary with the
        long_running_action D-17 field; returns TickResult.ok.

        lra = self._graph.query(actor, "current_long_action") or {}

        attention_state_pre = payload.get("attention_state") if isinstance(payload.get("attention_state"), dict) else None

        # 2. Single projection call (used by hook threshold eval AND observer)
        projection = self._projector.project_for(actor, attention_state=attention_state_pre)

        # 3. Run hook (advances turns_elapsed, evaluates thresholds, calls observer)
        hook_result = self._long_running_hook.process(
            actor=actor,

        sweep_nodes = self._run_passive_sweep(
            actor=actor,
            tick_id_str=tick_id_str,
            primary_mutations=[],
        )
        # Re-verify conservation across sweep (consistent with _handle_execute pattern)
        sweep_mutations = [m for node in sweep_nodes for m in node.mutations]
        if sweep_mutations:
            cons_verdict = self._conservation.verify(sweep_mutations)
            if cons_verdict.is_violation:

                # would interrupt the LRA tick. For v1, log a warning and continue — the
                # hook's threshold logic and the tick summary still reflect the truth of
                # this tick.

                    tick_id_str,

        post_lra = self._graph.query(actor, "current_long_action") if self._graph.has_node(actor) else None

            turns_total_for_summary = post_lra.get("turns_total")

                    "op": hook_result.fired_threshold.op,
                    "value": hook_result.fired_threshold.value,
                }
                if hook_result.fired_threshold is not None
                else None
            ),
            "interrupted": hook_result.interrupted,
        }

        # Build a synthetic ExecuteDecision for the summary so build_tick_summary's
        # decision-kind branching has something to dispatch on. Use mechanic_id
        # "continue_long_action" — diagnostics-friendly marker, never registry-resolved.
        from token_world.engine.models import ExecuteDecision
        synthetic_decision = ExecuteDecision(mechanic_id="continue_long_action")

        observer_in = getattr(self._observer, "last_input_tokens", 0) or 0
        observer_out = getattr(self._observer, "last_output_tokens", 0) or 0

        self._write_summary(
            tick_id_str=tick_id_str,
            action_text="[long_running_continuation]",
            decision=synthetic_decision,
            classified=None,
            trace=None,
            observation_text=hook_result.observation,
            start_time=start_time,
            classifier_in=0,
            classifier_out=0,
            observer_in=observer_in,
            observer_out=observer_out,
            long_running_action=long_running_action_field,
        )

            status="ok",
            action_text="[long_running_continuation]",
            decision_kind="execute",
            mechanic_id="continue_long_action",
            mutation_count=0,
            observation_text=hook_result.observation,
            yielded=False,
            refused=False,

        )

            tick_id=tick_id_str,
            observation=hook_result.observation or "",
